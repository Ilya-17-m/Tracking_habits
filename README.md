## Разработка

### Шаг 1. Установка зависимостей

#### Перед запуском проекта выполните устоновку зависимостей через инструмент poetry

##### Для начала установите сам poetry, напишите в терминале: ``pip install poetry``

##### Установка зависимостей, выполните команду в терминале: ``poetry install``



### Шаг 2. Запуск проекта

#### Чтобы запустить проект воспользуйтесь самым удобным варантом, для этого вам на устройстве понадабится установленный Docker и Docker Compose

#### Запустите все контейнеры через команду, которую напишите в терминале: ``sudo docker compose up --build`` тут контейнеры автомотически будут созданы и запущены

### Проект поддерживает онлайн маниторинг и анализа работы прилажения, в этом нам помогают сервисы: Sentry, Grafana, Grafana Loki, Prometheus.

#### Чтобы просматривать производительность роботоспасобности приложения перейдите по ссылке:

##### Grafana: ``http://localhost:3000`` тут сервисы Grafana Loki и Prometheus собирают метрики и отдают сервису Grafana для визуализации данных
