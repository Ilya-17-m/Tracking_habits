# Чат бот для трекинга привычек

### Чат бот имеет ряд функций позволяющих - создавать записи, редактировать и удалять их, а также напоминать о выполнении действий.

## Технологии

+ Python
+ FastAPI
+ SQLAlchemy
+ PostgreSQL
+ Alembic
+ PyTelegramBotAPI
+ Docker / Docker Compose
+ Sentry
+ Uvicorn
+ Poetry

## Разработка

### Чтобы запустить сервер backend состовляющей выполните команды:
``poetry run uvicorn backend.record_logic:app --port 8000``

### Чтобы запустить сервер акщтеутв состовляющей выполните команды:
``frontend.records_commands``
